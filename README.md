## ✏️ 프로젝트 소개
- 영화 종합순위를 크롤링하여 **영화 인기차트 정보** 제공
- 큐레이션을 통해 테마, 주제별 영화 **추천받기** 기능 제공
- 유저들간 영화 정보에 대해 이야기 하는 **게시판** 기능 제공
- **필터링**을 통한 **작품 탐색** 가능
- 영화를 **스크랩**하여 보고 싶은 영화 목록 모아두기
